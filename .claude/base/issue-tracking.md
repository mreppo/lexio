# Issue Tracking

Every action by every agent must be documented on the GitHub issue.

## Signatures

Since all agents post under the same GitHub user account, every comment MUST end with a signature block identifying which agent wrote it:

```
---
> 🤖 *<Agent Name> Agent (Claude CLI)*
```

### Agent Signatures

| Agent | Signature |
|-------|----------|
| Orchestrator | `> 🤖 *Orchestrator Agent (Claude CLI)*` |
| Developer | `> 🤖 *Developer Agent (Claude CLI)*` |
| QA | `> 🤖 *QA Agent (Claude CLI)*` |
| Reviewer | `> 🤖 *Reviewer Agent (Claude CLI)*` |
| DevOps | `> 🤖 *DevOps Agent (Claude CLI)*` |
| Release Manager | `> 🤖 *Release Manager Agent (Claude CLI)*` |
| Product Architect (Chat) | `> 🤖 *Product Architect (Claude Chat)*` |
