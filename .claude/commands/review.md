Review the current branch or specific files for code quality.

Usage: /review [file-paths]

This command runs a thorough code review using the reviewer agent.

Checks:
- TypeScript strict compliance
- Project conventions (CLAUDE.md)
- MUI theme usage
- StorageService abstraction compliance
- Accessibility
- Test coverage
- Code quality and naming

Use @agent-reviewer to perform this review.

The input is: $ARGUMENTS
