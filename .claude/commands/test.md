Write tests for specific files or features, or run and validate the full test suite.

Usage: /test [file-paths-or-feature-name]

This command delegates to the QA agent to:
- Write missing tests for the specified files/feature
- Ensure acceptance criteria are covered
- Run the full test suite and report results
- Identify untested edge cases

Use @agent-qa to perform this work.

The input is: $ARGUMENTS
